Istio Prometheus

  1. Check prometheus is created and available in istio-system namespace:

$ kubectl get svc/prometheus -n istio-system

  1. Create a tunnel for port 9090 via SSH:

$ ssh -L 9090:localhost:9090 root@<Master Node>

  1. Enable port forwarding:

$ kubectl port-forward svc/prometheus -n istio-system 9090:9090

  1. After this you can access http://127.0.0.1:9090/ from your local PC.

  2. Navigate to Status > Targets and check that all targets are discovered and their statuses are monitored.

  3. Navigate to Status > Service Discovery and check that all services are discovered and their statuses are monitored.

  4. Navigate to Graph.

  5. Choose any command you want and select it.
